Draw

The draw tool allows you to click and drag the mouse on the canvas to paint.
The colour that is draw depends on your current selected colour (more info in the colours tab)
Drawing on the canvas will draw to the current selected layer which will be layer 1 if none have been added.
The size of the draw tool and the lines it creates depends on your current brush size.

 

Erase

The erase tool allows you to remove any colour you have painted to the canvas e.g. via the draw tool or shapes tools. The erase tool will only erase on the current selected layer. The size of the erase tool is also dependant on the current brush size.

 

Clear

The clear tool will completley reset the selected layer to empty. The clear tool will remove any brush strokes or colour on the canvas so be carefull when using this tool. This action cannot be undone.

 

Draw Line / Shapes

This section covers information for the draw line, rectangle, triangle and circle tools. All of these tools will draw to the selected layer in the current selected colour. The draw line tool simply draws a perfectly straight line from where you click the mouse to where you release it on the canvas. The draw triangle tool will draw a triangle between the three points you click on the canvas. The draw circle tool will draw a circle using the points where you click and release the mouse on the canvas. The draw rectangle tool will draw a rectangle using the points where you click and release the mouse on the canvas.

 

Select Colour

The select colour tool allows you to change your active colour to one which is already drawn to the canvas. To use it simply click the tool from the menu and then click the pixel which has the colour you wish to select.

 

Brush Customization

The brush size can be altered via the slider or tool buttons. The slider allows you to quickly set the brush size to bigger or smaller. For more precise modifications you can use the buttons in the tools menu to change the width and height of the brush.
