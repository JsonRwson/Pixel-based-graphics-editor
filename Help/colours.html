Quick Select Colours

The program has a grid of quick select colours for ease of access to frequently used colours e.g. primary colours. These cannot be changed.

 

Used Colours

The program will keep track of the last 3 selected colours used. This is usefull if you want to quickly switch back to a previous colour you have used. These colours are shown below the quick select colour grid.

 

Colours and Tools

All tools that draw to the canvas will draw in your currently selected colour.

 

Custom Colours

Custom colours can be selected with specific rgb values. Simply click the colour wheel button and use the sliders to change the r,b,b values.
